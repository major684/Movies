------------------------------------------------------------Contents----------------------------------------------------------------
media.py: 			This file contains the class Movie which holds all the properties of our movie class
entertainment_center.py: 	This is our main file that calls all the other files and generates the final html file.
fesh_tomatoes.py: 		This python file generates html and then dynamically puts each object from the entertainment center
				file into the final html product.
fresh_tomatoes.html: 		This is the final web product that we see all our movies in the browser.
-------------------------------------------------------------------------------------------------------------------------------------


---------------------------------------------------------Instructions----------------------------------------------------------------
1. Download all files
2. Ensure you have Python installed
3. Open the command prompt and navigate to where you downloaded the files
4. Run the file entertainment_center.py
5. You should have your default browser open up with the list of movie trailers

NOTE: To make any changes to the type of movies that show on the html file please open up the entertainment_center.py file in notepad 
      or another IDE.  You will then change each each object instance with a movie of your choosing.  Happy Coding!!
